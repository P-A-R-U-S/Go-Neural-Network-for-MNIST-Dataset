<h3 align="center">Goland project to design neural network using <a href="https://en.wikipedia.org/wiki/MNIST_database">MNIST (Modified National Institute of Standards) dataset </a> </h3>

MNIST stands for Modified National Institute of Standards and Technology dataset, and it’s a collection of 70,000 grayscale images of handwritten digits (0 through 9). 
Each image is a tidy 28x28 pixels, and it comes with a label telling us which digit it is.

The dataset is split into two parts:
- Training Set: 60,000 images to teach our neural network.
- Test Set: 10,000 images to check how well it learned.

MNIST is like the "Hello World" of machine learning—simple, well-organized, 
and widely used. It’s perfect for our first adventure into building a neural network from scratch.
